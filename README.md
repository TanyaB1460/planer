# Planer - Планировщик задач

Одностраничное веб-приложение для управления задачами на чистом HTML, CSS и JavaScript.

## Функционал

- Добавление новых задач с мгновенным обновлением интерфейса
- Отметка задач как выполненных через чекбоксы
- Удаление отдельных задач
- Очистка всего списка с подтверждением
- Статистика: общее количество, выполненные, оставшиеся задачи
- Фоновая синхронизация с JSONPlaceholder API через POST-запросы
- Автоматическое сохранение в LocalStorage
- Система уведомлений о действиях
- Полностью адаптивный дизайн
- Защита от XSS-атак

## Запуск

1.Откройте `index.html` в браузере

Приложение работает офлайн с автоматической синхронизацией.

## Технические особенности

- Код организован в один класс PlanerApp с отдельными функциями для каждой задачи
- Обработчики событий работают для новых элементов, добавленных динамически
- Мгновенное добавление задач без ожидания сетевого ответа
- POST-запросы отправляются через fetch с обработкой ошибок
- Все данные сохраняются в браузере (LocalStorage)
